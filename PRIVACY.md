# Privacy

PPTX Viewer processes PowerPoint files (`.pptx`, `.ppsx`, `.pptm`, `.ppsm`, `.potx`, `.potm`) locally inside your editor (VS Code, Cursor, Windsurf, etc.).

## Data collection

- No telemetry is collected by this extension.
- No personal data is intentionally collected by this extension.

## Network access

- The extension does not send your file contents to external servers.
- The extension does not require network access for normal operation.

## Logs

- If an error occurs, the editor may show logs in `Output` and/or Developer Tools.
  These logs can include filenames and error details.
