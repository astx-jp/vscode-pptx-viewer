# Privacy

PPTX Viewer processes PowerPoint (`.pptx`) files locally inside your VS Code environment.

## Data collection

- No telemetry is collected by this extension.
- No personal data is intentionally collected by this extension.

## Network access

- The extension does not send your `.pptx` contents to external servers.
- The extension does not require network access for normal operation.

## Logs

- If an error occurs, VS Code may show logs in `Output` and/or Developer Tools.
  These logs can include filenames and error details.
